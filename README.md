## 🔥 V2.1 重大更新 (2025.03)

本次更新解决了“创建变慢”的核心痛点，并带来了期待已久的个性化功能：

### 1. ⚡️ Excel 引擎性能重构
- **极速响应**：不再全量扫描历史数据，改为**智能回溯**算法。仅处理最近一天的旧数据，即使表格有几千行，创建文件夹也**秒开秒关**。
- **智能填表**：
  - **无条件追加**：彻底移除重复检测逻辑，确保每一条输入都能准确写入表格。
  - **智能序号**：自动识别输入文本中的序号（如 `1. 张三...`），不再傻傻地自动递增。
  - **当月套数**：自动读取上一行数据并递增，数学不好的也不怕了。

### 2. 🎨 表格视觉标准化
- **红绿灯配色**：
  - **今日新数据**：HS 列标**黄**（醒目提醒）。
  - **历史旧数据**：整行标**绿**（表示已完成）。
  - **智能留白**：自动跳过 **K、L、M、N** 四列（第 11-14 列），保留原始样式，不破坏表格美感。

### 3. 👤 摄影师姓名自定义
- **不再只是“贺志”**：主界面新增**摄影师姓名**输入框。
- **全局生效**：
  - 文件夹名自动变为 `MMDD{你的名字}`。
  - Excel 文件名自动变为 `MMDD{你的名字}.xlsx`。
  - 表格内“摄影师”列自动填充你的名字。
- **智能记忆**：
  - 首次打开默认为空（保持清爽）。
  - 一旦输入，自动保存至本地配置（`~/.fangkan_helper_config.json`），下次打开自动加载，无需重复输入。

## ✨ 项目简介

房堪拍摄的真实日常通常是这样的：

- 今天几套？  
- 相片、VR 两路目录要不要对齐？  
- 日期对没对？  
- 文件名里有没有“/”？  
- 导卡的时候界面卡死了？  
- 拍完一堆，心里只剩一句：“我这是在干嘛？”

本工具正是为了解决这些“看起来不复杂、但每天都在消耗你精力的小事”。

它是一个**跨平台桌面 GUI 应用**，专注于三件事：

1. **文件夹一次建对**
2. **素材一次导完**
3. **收入算得清清楚楚**

而 V2.1 版本，让这三件事变得**更快、更稳、更好看**。

## 🚀 核心功能总览

### 📂 1. 批量文件夹生成 批量创建文件夹（房堪刚需）

- **一次输入，双路同步**
  - 同时在「相片」与「VR」目录中创建对应房源文件夹
- **日期自动识别**
  - 年 / 月 / 日自动匹配，无需手敲
- **标准化目录结构**
  - 相片目录：  
    `.../{YYYY}相片/{MM}月/{MMDD}{摄影师姓名}`
  - VR 目录：  
    `.../{YYYY}VR/{MM}月/{MMDD}`
- **非法字符自动清洗**
  - `/ \ : * ? " < > |` 自动替换，macOS / Windows 通吃
- **智能文本清洗**
  - 实时去除门店名称前后缀（如 `XHJ`、`A店A业务组`）
  - 只保留关键门店名，输入框实时预览
- **高频操作一键完成**
  - 打开目录 / 复制路径 / 示例填充，全都不需要多想

---

### 📊 2. Excel 自动继承与智能填充

- **自动回溯**
  - 创建文件夹时，自动查找最近一天（365天内）存在的 Excel 文件
- **智能复制**
  - 将昨日的 Excel 模板自动复制到今日文件夹，并重命名。
- **样式自动应用**
  - 自动为新旧数据应用不同的背景色与字体颜色，一目了然。

---

### 💰 3. 收入实时计算（精神支撑系统）

- 按 **¥28 / 套** 自动计算今日预计收入
- 输入即统计，创建即更新
- 完成后弹窗提示：  
  > “今天不亏，至少是紫的。”

---

### ⚡️ 4. 极速智能导卡

V2 引入了真正**可感知的性能提升**：

#### 🚄 多线程并发导入
- 后台多线程处理文件 IO
- **导卡过程中界面不冻结**
- 相片 / VR **双路并行传输**

#### 📊 极客级进度可视化
- 实时传输速度显示（如 `86.4 MB/s`）
- 双进度条独立显示：
  - 百分比
  - 文件数（`23 / 51`）
- 你能清楚知道：
  - 在干活
  - 干得快
  - 没在摸鱼

#### 🛡 安全机制
- 存储卡未插入 → 自动跳过
- 源目录为空 → 不报错、不吓人
- 所有路径均做存在性校验

## 📦 打包指南

如果需要生成独立的 `.app` 或 `.exe` 文件（无需安装 Python 环境即可运行）：

1. **安装 PyInstaller**
   ```bash
   pip install pyinstaller
   ```

2. **执行打包命令**
   ```bash
   # macOS
   pyinstaller --name "房堪助手" --windowed --icon "assets/icons/final_icon.icns" --add-data "src:src" --add-data "assets:assets" --hidden-import "openpyxl" --clean --noconfirm "房堪助手.spec"
   ```
   打包完成后，应用程序将生成在 `dist/` 目录下。
